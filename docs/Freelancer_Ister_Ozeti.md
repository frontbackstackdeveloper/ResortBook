# Freelancer İlan İsterleri Özeti

Seçilen ilan: WordPress Resort Website With Booking
Platform: Freelancer

İlandan alınan ana isterler:

- Resort / otel için web sitesi
- Booking / rezervasyon sayfası
- Online rezervasyon formu
- Real-time availability calendar
- Secure payment processing
- Oda müsaitliği ve envanter kontrolü
- Double-booking engelleme
- Misafire ve admine otomatik e-posta bildirimi
- Contact page

Bu projede WordPress yerine ASP.NET Core MVC + SQLite + EF Core kullanılacaktır. Ödev yönergesinde müşterinin istediği teknolojiyi kullanma zorunluluğu bulunmadığı için teknoloji uyarlaması yapılmıştır.
